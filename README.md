# Portfólio | Kauan Bastos

Site pessoal de portfólio com projetos, habilidades e contato.

## Tecnologias

- React
- Vite
- Tailwind CSS
- Lucide Icons
- Radix UI

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy (GitHub Pages)

```bash
npm run deploy
```
